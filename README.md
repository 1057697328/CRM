# CRM

### 	前端所用技术:
Vue2 + Vuex + Vue Router + Element UI + ES6 + webpack + axios + Node.js

### 后端所用技术:
* Spring
* SpringMVC
* MyBatis
* MySQL
* Shiro：安全框架
* Swagger2：Api文档生成
* Nginx
* Tomcat
* Maven
* 其它开发工具
	* Jenkins：持续集成
	* 阿里JAVA开发规约插件
